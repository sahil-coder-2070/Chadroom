```md
# Chadroom

Chadroom is a small practice project built to work on UI and motion-based interactions using React.  
The goal is to practice layout, animations, and basic UI behavior.

## Purpose

- Practice React UI structure
- Add simple animations using motion
- Work on micro interactions
- Style components with Tailwind CSS

## Tech Stack

- React
- Motion
- Tailwind CSS
- Vite
- Bun
- npm

## Features

- UI-only project
- Simple motion animations
- Micro interactions on UI elements
- Responsive layout

## Getting Started

Install dependencies:

npm install

Start the dev server:

npm run dev

Open the local URL shown in the terminal.

## Project Status

This is a practice and learning project.  
Not intended for production use.

## Notes

- No backend
- UI only
- Built for learning
```
